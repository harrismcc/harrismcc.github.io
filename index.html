SeamCarving

Seam carving, as outlined in this paper, is a process by which the proportions of an image can be changed without cropping or squishing/stretching. The algorythm works by finding a connected line of pixels, called a seam, that travels along an area of the image deemed least important. This is detirmined by the amount of difference each pixel has to its neighbors. Pixels that "won't be missed" are removed from the image in order to preserve the important parts but still shrink the image.

Visualize Julia Sets

Julia sets are fractals of imaginary numbers iterated n number of times according to the following equation: z = z^2 + c where z and c are imaginary numbers. By plotting the real and imaginary parts of these numbers the sets can be visualized. By changing the starting value of the imaginary number different Julia sets can be represented.

Huffman-Compression

Huffman compression is a method of lossless data compression. Developed by David Huffman in the paper "A Method for the Construction of Minimum-Redundancy Codes" My code essentially works by building a recursive tree based on the frequency of the letters as they appear in the origional text. I then encode that tree as binary. e.g. the "left" move is a one and the "right" move is a zero. Thus if the route to the letter "S" is left, right, right; then the binary string for that letter will be 100. The code then applies this to all letters to create one long binary string. This string is then broken up into pieces of size 8 and made into unicode charicters
